# Noteflux

A modern full-stack blogging platform built with React, Node.js, Express, MySQL, and TailwindCSS.

Noteflux allows users to create, explore, and manage blogs through a clean and modern SaaS-style interface. The platform includes authentication, blog management, comments, profile editing, and responsive UI optimized for a smooth user experience.

---

## 🚀 Features

### Authentication
- User registration and login
- JWT authentication
- Protected routes
- Secure password hashing

### Blogging System
- Create blogs
- Edit blogs
- Delete blogs
- Explore all blogs
- Single blog detail page

### Comments System
- Add comments on blogs
- View all blog comments
- User-linked comments

### Profile Management
- Update profile information
- Upload profile photo
- Add bio and phone number

### UI/UX
- Modern SaaS-inspired design
- Fully responsive layout
- Clean typography and spacing
- Smooth hover animations
- Reusable component architecture
